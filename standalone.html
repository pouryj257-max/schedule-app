📅 Google 캘린더 연동

1. 아래 'ICS 파일 다운로드' 버튼을 눌러 파일을 저장합니다.
2. Google 캘린더 (calendar.google.com) 접속
3. 설정 → 가져오기 및 내보내기
4. 다운로드한 .ics 파일을 선택하여 가져오기

📱 네이버 캘린더 연동

1. 아래 'ICS 파일 다운로드' 버튼을 눌러 파일을 저장합니다.
2. 네이버 캘린더 (calendar.naver.com) 접속
3. 설정 → 일정 가져오기
4. 다운로드한 .ics 파일을 선택하여 가져오기

🍎 iPhone 캘린더

1. 아래 'ICS 파일 다운로드' 버튼을 누르면 자동으로 캘린더 앱에 추가됩니다.
2. 또는 파일을 이메일로 전송 후 iPhone에서 열어 추가합니다.

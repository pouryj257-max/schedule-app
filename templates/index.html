외부 캘린더 연동

아래 iCal URL을 복사하여 Google 캘린더, 네이버 캘린더 등에 등록하면 일정이 자동으로 연동됩니다.

Google 캘린더 연동 방법

  1. Google 캘린더 열기
  2. 좌측 "다른 캘린더" 옆 + 클릭
  3. "URL로 구독" 선택
  4. 위 URL 붙여넣기

네이버 캘린더 연동 방법

  1. 네이버 캘린더 열기
  2. 설정 > 캘린더 가져오기
  3. "URL로 가져오기" 선택
  4. 위 URL 붙여넣기
